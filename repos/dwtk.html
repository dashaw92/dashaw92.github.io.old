DWTK

DWTK is a toolkit that provides a set of widgets (buttons, textboxes, listboxes, and more..) to the developer. With this toolkit, the developer no longer needs write all of the widgets from scratch.
DWTK provides an event API for all widgets (with the exception of the built in label) so that advanced applications can be implemented in minimal time.
If the developer needs a custom widget that is not provided by default, a new widget can be created that works like all the others.
DWTK uses java.awt.{Canvas, Graphics} and can be used on top of a current application.
